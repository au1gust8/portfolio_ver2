Html5 Web Project

한국디자인진흥원 기업형 웹사이트

프로젝트 소개

기존 한국디자인진흥원 웹사이트의 주요기능 전달의 결핍과 접근성의 부재를 문제 삼아 트렌드에 맞는 UI Design과 웹표준을 지향하는 마크업으로 웹 접근성을 고려하여 새롭게 구축하였습니다.

기존 페이지 문제점

기존 웹사이트의 정돈되지 않은 Interface.

누락된 네비게이션 등으로 W3C검사 시 웹표준에 어긋남

용도가 불분명한 빈 a태그와 버튼 태그. 이미지로 처리하고 속에 기능이 비어 있는 경우 다반수

Killer Function인 정보제공이 사용자에 대한 편의를 고려하고 있지 않음

개선방향

웹표준성과 접근성을 고려한 크로스브라우징

다량의 정보공지형 사이트를 역할에 충실할 수 있도록 정돈화

새롭게 정돈한 인포메이션 아키텍쳐

이미지처리 된 본문을 텍스트 전달
